# Requirements

+ PHP 8+
+ PHP Extensions:
    + [ext-curl](http://php.net/manual/en/book.curl.php)
    + [ext-sodium](http://php.net/manual/en/book.sodium.php) (api client don't need this)

# Installation

```bash
 git clone https://github.com/iota-community/iota.php
```
